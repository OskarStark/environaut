Class

Environaut\Config\Reader\Dom\DomElement

class DomElement extends DOMElement implements IteratorAggregate

Custom \DOMElement implementation for Environaut to have some convenience methods to get child nodes etc.

Please be aware, that
by default these methods use the Environaut default namespace URI.
