Environaut\Checks\ComposerSecurityCheck

class ComposerSecurityCheck extends Check

The SensioLabs Security Checker is a command line tool that checks if your application uses dependencies with known security vulnerabilities.

It uses the SensioLabs Security
Check Web service and the Security Advisories Database behind the scenes.
